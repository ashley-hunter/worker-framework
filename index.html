About Worker Framework

The Worker Framework provides a foundation for building cross-platform, cloud-ready, distributed data-processing microservices (workers).

Fault Tolerant Messaging

High Availability and Fault Tolerant Messaging.

External Data Exchange

Data exchange with pluggable storage.

Configuration Management

Flexible configuration management with pluggable sources.

Health Monitoring

Failure recovery support with integrated health reporting.

Serialization and De-serialization

Serialization and De-serialization of Message Payload.

Bulk Message Handling

Bulk Message Handling with flexible data partitioning.
